# AB Apps 2

Full-stack app with a Django REST backend and a React + Vite frontend.

## Overview

- Backend: Django REST API with JWT auth, clients, places, documents, and work orders
- Frontend: React + TypeScript + Vite application

## Documentation

- [Backend setup and API docs](backend/README.md)
- [Frontend setup and development docs](frontend/README.md)

## Quick Start

### Backend

1. Go to `backend/`
2. Create a virtual environment
3. Install dependencies
4. Copy + modify `.env.sample`
5. Run migrations
6. Start the server

### Frontend

1. Go to `frontend/`
2. Copy + modify `.env.sample`
3. Install dependencies
4. Start the dev server
